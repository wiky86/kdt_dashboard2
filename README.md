# 유비온 디지털교육센터 - 훈련 과정 데이터 관리 시스템

훈련 과정 데이터를 검색, 필터링, 분석할 수 있는 웹 애플리케이션입니다.

## 🚀 주요 기능

### 📊 데이터 검색 및 필터링
- **기본 검색**: 기관명, 과정명으로 검색
- **고급 필터**: NCS직무, 훈련시간, 시간당훈련비, 수료율, 취업률, 만족도, 개강일 범위
- **실시간 검색**: 구글 시트와 연동하여 최신 데이터 제공

### 📈 통계 분석
- **시간별 분석**: 월별, 분기별, 반기별, 연도별 통계
- **지표별 분석**: 모집률, 취업률, 수료율, 만족도, 모집인원 등
- **필터링 분석**: 전체 과정, 검색어별, NCS직무별 분석
- **시각화**: Chart.js를 활용한 동적 차트 생성

### 📋 데이터 관리
- **상세 정보**: 개별 훈련 과정의 회차별 상세 정보
- **데이터 내보내기**: Excel(CSV), PDF 형식으로 내보내기
- **통계 내보내기**: 차트 이미지 다운로드

## 🛠 기술 스택

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **차트 라이브러리**: Chart.js
- **아이콘**: Font Awesome
- **데이터 소스**: Google Sheets API
- **스타일링**: shadcn/ui 기반 커스텀 CSS
